Loops

        LOOPS:
    

  • Loops can execute a block of code a number of times. There are multiple kinds of Loops
which are executed at the tiime of the different requirements. 
    • 

  • JavaScript supports different kinds of loops:
      • 

    1. for - loops through a block of code a number of times
      2. 
for (expression 1; expression 2; expression 3) {
    // code block to be executed
  }

    2. for/in - loops through the properties of an object
      3. 
for (key in object) {
    // code block to be executed
  }

    3. for/of - loops through the values of an iterable object
      4. 
for (variable of iterable) {
    // code block to be executed
  }

    4. while - loops through a block of code while a specified condition is true
      5. 
while (condition) {
    // code block to be executed
  }

    5. do/while - also loops through a block of code while a specified condition is true
    
do {
    // code block to be executed
  }
  while (condition);

  • Loops can execute a block of code as long as a specified condition is true.
    •

Examples

    Example-1

  (Showing for loop:)
const cars = ["BMW", "Volvo", "Saab", "Ford", "Fiat", "Audi"];
let text = "";
for (let i = 0; i < cars.length; i++) {
  text += cars[i] + "
";
}
document.getElementById("demo").innerHTML = text;


    Example-2

  (Showing for-in loop:)
  const person = {fname:"hiya", lname:"smith", age:25}; 

  let txt = "";
  for (let x in person) {
    txt += person[x] + " ";
  }
  
  document.getElementById("demo").innerHTML = txt;

 

    Example-3

  (Showing for-of loop:)
    const fruits = ["kiwi", "berry", "grapes"];
    
    let text = "";
    for (let x of fruits) {
      text += x + "
";
    }
document.getElementById("demo").innerHTML = text;



    Example-4

 (Showing while loop:)
 let text = "";
let i = 0;
while (i < 10) {
  text += "The number is " + i;
  i++;
}
document.getElementById("demo").innerHTML = text;



    Example-5

  (Showing do-while loop:)
  let text = ""
let i = 0;
do {
  text += "
The number is " + i;
  i++;
}
while (i < 10);
