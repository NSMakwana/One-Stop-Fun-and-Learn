The Script Tag

In HTML, JavaScript code is written between script tag.

For example:

<"script">
document.getElementById("demo").innerHTML="My First JavaScript";
<"/script">

JavaScript Functions and Events

A JavaScript function is a block of JavaScript code, that can be executed when it is "called" for.

For example, a function can be called when an event occurs, like when the user clicks a button.

JavaScript in <"head"> or <"body">

We can place any number of scripts in an HTML document.

Scripts can be placed in the <"body">,or in the <"head"> section of an HTML page, or in both.

JavaScript in <"head">

In the following example, a function is placed in the <"head"> section of the page.
The function will be invoked when a button is clicked.

Example:

<"html">
<"head">
<"script">
function myfunction() {
document.getElementById("demo").innerHTML="This project is basically about doing fun and learn together at the same time.";
}
<"/script">
<"/head">
<"body">
<"h2">Example of JavaScript in the head section<"/h2"> <"p id= demo">This is our 1st project.<"/p">
<"button type=button"
onclick="myfunction()">
Try it<"/button">

<"/body">
<"/html">
