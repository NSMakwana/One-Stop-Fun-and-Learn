Output

Possibilities of display in JavaScript

The different possibilities of displaying the outputs are:

  • Writing directly into an HTML element, using innerHTML method.
  • Writing into the HTML output using document.write() method.
  • Writing into an alert box, using window.alert() method.
  • Writing directly into the browser console, using console.log() method.

    • Using innerHTML method

    To access any HTML element, JavaScript can use document.getElementByID() method.

    The id attribute defines the HTML element. The innerHTML property defines the HTML content.

    For Example:

    <"html">
    <"body">
    <"h2"> This is our first project. <"/h2">
    <"p"> This game is about teaching JavaScript. <"/p">
    <"p id=demo"><"/p">
    <"script">
    document.getElementByID("demo").innerHTML=5+6;
    <"/script">
    <"/body">
    <"/html">

    Using document.write() method

    For doing any testing, it is very convenient to use document.write() method.

    <"p"> This game is about teaching JavaScript. <"/p">
    <"script"> document.write(5+6); <"/script">
    <"/body">
    <"/html">

    For Example 2:

    <"html">
    <"body">
    <"h2"> This is our first project. <"/h2">
    <"p"> This game is about teaching JavaScript. <"/p">
    <"button type=button onclick=document.write(5+6)">Try it<"/button">
    <"/body">
    <"/html">

    More about window.alert() method

    We can also skip the keyword window

    In JavaScript, the window object is the global Scope object. It means that variables, properties and methods by default belong to the window object. It also means that specifying the window keyword is optional.

    For Example:

    <"html">
    <"body">
    <"h2"> This is our first project. <"/h2">
    <"p"> This game is about teaching JavaScript. <"/p">
    <"script"> alert(5+6); <"/script">
    <"/body">
    <"/html">

    Using console.log() method

    For debugging purposes, we can call console.log() method to display the data.

    For Example:

    <"html">
    <"body">
    <"script">
    console.log(5+6);
    <"/script">
    <"/body">

    Using JavaScript Print method

    JavaScript does not have any print object or print methods.

    You cannot access output devices from JavaScript.

    The only exception is that we can call window.print() method in the browser to print the content of the current window.

    For Example:

    <"html">
    <"body">
    <"button onclick=window.print()">Print this page<"/button">
    <"/body">
    <"/html">
